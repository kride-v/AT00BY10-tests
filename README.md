# Student template

[![Coverage Status](https://coveralls.io/repos/github/kride-v/AT00BY10-tests/badge.svg?branch=main)](https://coveralls.io/repos/github/kride-v/AT00BY10-tests?branch=main)

## Purpose of this repository

This is a project template for students participating in the Software Testing course
at LAB University of Applied Sciences.

The repository only contains the source code that is under testing, `package.json` skeleton,
and LICENSE file.

Source code folder contains a separate license file that must **NOT** be removed under any circumstances!
Removing this license file directly violates terms and conditions of the software under testing.
Individuals who remove or modify the license file will also carry the consequences.

## Testing and Coverage

This repository includes unit tests for the library. Test coverage reports are automatically
generated using Jest and sent to Coveralls via GitHub Actions. The badge above shows
the current test coverage of the main branch.
